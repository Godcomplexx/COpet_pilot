# ADR-004: Outdoor Mode отложен

## Статус

Принято — 2026-07-16.

## Контекст

Outdoor Mode требует GPS/GNSS, другой модели питания, корпуса, outdoor-тестов и
дополнительного UI. Эти задачи не доказывают качество базового настольного
companion-device и расширяют аппаратный риск.

## Решение

Outdoor Mode, GPS, скорость, дистанция и activity log исключаются из MVP и
текущего меню. Они остаются направлением следующей аппаратной версии.

## Последствия

- текущая архитектура не создаёт GPS driver/task и outdoor states;
- MPU6050 используется только для настольных реакций и пробуждения;
- UART и пины GPS пока не резервируются окончательно;
- после завершения Desk/Focus/Assistant/Mini TV создаётся отдельное ТЗ Outdoor.

## Пересмотреть решение, если

- настольный MVP стабилен и имеет корпус;
- выбран GNSS-модуль и модель питания;
- outdoor-функция подтверждена отдельным пользовательским сценарием.
