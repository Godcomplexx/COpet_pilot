# Media assets for the README

Drop the following files here and they will appear in the top-level `README.md`.
Keep them small (GIFs under ~5 MB) so the page loads fast.

| File | What to capture |
|---|---|
| `copet-hero.gif` | Hero shot: the device on a desk, face blinking / reacting |
| `desk-mode.gif` | Desk Mode: blink, gaze, touch reaction, room data |
| `focus-mode.gif` | Focus Mode: preset selection with the encoder, timer running |
| `prototype.jpg` | Clear photo of the assembled hardware prototype |

Tips: film at the display's native framerate, crop tight to the screen for the
mode GIFs, and use even lighting for the hardware photo.
