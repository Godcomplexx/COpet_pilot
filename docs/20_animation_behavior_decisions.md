# CoPet Pilot — таблица эмоций и анимаций

Статус: предлагаемая таблица поведения v1. Прошивка пока не изменена.

## Правила выбора состояния

| Приоритет | Назначение | Правило |
|---|---|---|
| P0 | безопасность и критическое состояние | прерывает любую декоративную анимацию |
| P1 | касание, движение и текущая системная операция | прерывает P2–P4 |
| P2 | режим, температура, влажность, питание | действует, пока сохраняется условие |
| P3 | спокойное фоновое занятие | запускается только при бездействии |
| P4 | редкая пасхалка | не чаще одного раза за длительный период |
| M | ручной запуск | только из галереи или диагностики |

Обозначения хранения:

- **ГОТОВО** — уже есть в текущей прошивке;
- **FLASH** — переносим во внутреннюю память ESP32-WROOM-32;
- **FLASH-2** — переносим после основного набора, если хватает памяти;
- **SD** — тяжёлая анимация, добавляется после установки SD-карты;
- **ГАЛЕРЕЯ** — автоматически не запускается.

## Временная шкала спокойного Desk Mode

| Время бездействия | Поведение |
|---:|---|
| 0–30 с | `neutral`/контекстная эмоция, моргание и спокойные взгляды |
| 30–90 с | `bored`, более медленные взгляды |
| после 90 с | короткое занятие P3 длительностью 8–20 с |
| далее каждые 1–3 мин | новое уместное занятие без недавних повторов |
| после 10 мин | `sleepy`; активности становятся реже |
| после 15 мин ночью | `snore` до касания или движения |
| после 30 мин | появляется шанс редкой пасхалки P4, если нет предупреждений |

Контекстные наборы для случайного выбора:

| Условие | Предпочтительные состояния |
|---|---|
| утро | `clock`, `weather`, `wired`, `happy` |
| день | `dice_roll`, `snake`, `game_of_life`, `ping_pong` |
| вечер | `chill`, `campfire`, `smoking`, `cat`, `zen` |
| ночь | `sleepy`, `snore`; очень редко `matrix` или `blackhole` |
| прохладно | `tired`, `shiver`, `campfire` |
| жарко | только приоритетный `scared + overheated`, без фоновых занятий |
| нет сети | локальные игры; не выбирать сетевые действия и виджеты |
| идёт Focus | `focused`, `working`, `waiting`, `target_lock/release` |

## Защита от повторов и конфликтов

1. P0 всегда прерывает P1–P4.
2. Касание, энкодер или движение немедленно завершают P3/P4.
3. Из случайного набора исключаются четыре последних занятия.
4. Одно занятие не запускается повторно раньше чем через 10 минут.
5. `smoking` запускается не чаще одного раза в час.
6. Любая P4-пасхалка запускается не чаще одного раза за 6 часов.
7. Автоматическая анимация не блокирует меню, датчики и системные задачи.
8. Ошибка SD или ресурса возвращает `neutral` и не перезагружает устройство.

## Первый набор без SD-карты

Первая очередь расширения внутренней прошивки:

1. Эмоции: `alert`, `attentive`, `kawaii`, `awe`, `focused`, `sad`, `worried`,
   `nervous`, `angry`, `confused`, `disoriented`, `standby`.
2. Жесты: все 12 жестов.
3. Реакции: `acknowledge`, `laugh`, `nod`, `pop`, `refuse`, `squint`.
4. Системные действия: `connecting`, `listening`, `thinking`, `searching`,
   `processing`, `scanning`, `waiting`.
5. Эффекты: `boot_draw`, `recording`, затем `campfire`, `zen`, `dice_roll`,
   `snake`, `thermal_scan`, `target_lock`, `target_release`.

После оценки размера прошивки и FPS добавляется очередь `FLASH-2`. Эффекты `SD`
не переносятся во внутреннюю память только ради количества.
